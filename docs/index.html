ROBOTEST

OpenSource DevOps End2End Test Engine based on Selenium and RestAssured for Java projects

Made by developers, for developers

Developers are best actors to develop and maintain E2E tests. ROBOTEST simplifies development of E2E test in your Java projects 

Minimizes E2E testing effort

ROBOTEST supports browsers enhaced by Selenium WebDriver, like Chrome, Firefox, Internet Explorer, Edge, etc. BrowserStack add supports for E2E testing in Windows, iOs and Android platforms in this browsers or in native mobile apps. Docker enables headless execution for Chrome and Firefox in  *NIX environments. ROBOTEST simplify and minimize the effort to maintain different lines of development for different devices and channels
